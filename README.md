## 搬瓦工 VPS 升级是什么意思？先搞清楚概念

搬瓦工的 VPS 套餐升级，指的是把你当前在用的套餐换成配置更高的方案——比如内存从 1GB 升到 2GB、月流量从 1TB 升到 2TB，或者从普通 KVM 线路升到 CN2 GIA 线路。

升级不用重装系统，不会换 IP，原来的数据和配置全部保留，只需要补交新旧套餐之间的差价就行。这是搬瓦工设计得还算合理的一个功能，省去了重新买机器再迁移数据的麻烦。

## 搬瓦工VPS无法升级的五种常见情况

这是核心部分。遇到升级问题，先对照下面这五种情况逐一排查。

### 情况一：走错页面了，KiwiVM 面板里找不到升级入口

很多人第一次升级，直接进了 KiwiVM 控制面板（就是管 VPS 系统的那个面板），在里面找了半天升级按钮——找不到是正常的，因为升级操作不在 KiwiVM 里，在**客户中心后台**。

两个后台别搞混：
- **KiwiVM 面板**：管 VPS 内部，重启、重装系统、切换机房、查流量
- **客户中心（bwh81.net 或 bwh88.net）**：管账户和套餐，升级、续费、提交工单都在这里

升级的正确入口在客户中心 → Services → My Services → 找到对应的 VPS → 齿轮图标 → **Upgrade**。

### 情况二：当前套餐已经是最高配，无更高等级可选

进了升级页面，发现列表里没有任何可选方案，或者所有方案都比当前套餐低——这说明你当前用的已经是该系列里的顶配了，自然没有可以升的方向。

这种情况下，如果确实需要更好的配置，只能考虑直接购买新的高端套餐，比如从普通 CN2 GIA-E 跨到香港或日本的高端线路。

### 情况三：套餐系列不同，无法跨系列直接升级

搬瓦工的套餐分几个系列，不同系列之间有升级限制：

- 普通 KVM 套餐 → 可升级到更高的 KVM 或 CN2 GIA 系列
- CN2 特价促销版 → 只能升级到同系列更高配，不能升级到普通 KVM
- 限量版套餐（如 BOX 系列）→ 升级规则较特殊，部分需提交工单

说实话，这个限制以前困扰过挺多人。问题是，升级页面上经常只显示套餐名称和价格，不显示详细配置，所以不知道那个套餐到底是什么系列。建议先在官网套餐页对照一遍配置，再回来选。

### 情况四：该功能曾一度暂停，但现已恢复自助升级

搬瓦工历史上有一段时间确实取消了自助升级功能，那时候需要提交工单让客服手动操作。后来因为需求量大，又重新开放了自助升级。

如果你在网上看到的文章写"必须提工单才能升级"——那篇文章可能比较老，现在大部分套餐已经支持后台直接操作。

不过降级这边还没完全放开：**降级目前仍然需要联系客服处理**，不能自助完成。

### 情况五：账户存在异常，升级入口被屏蔽

这个情况比较少见，但确实有。账单信息不完整、付款记录异常、或者 VPS 当前处于暂停（Suspended）状态，都可能导致升级入口无法正常显示。

排查方法：先检查账户里有没有未处理的通知邮件，再登录后台看 VPS 当前状态是不是 Active。如果 VPS 被暂停，先把暂停问题解决（可能是流量超额、CPU 占用过高、账单问题），再操作升级。

## 正常情况下，搬瓦工VPS升级的完整步骤

确认不是上面那几种异常情况后，按以下步骤操作：

1. **登录客户中心**：访问 bwh81.net 或 bwh88.net，右上角点击 Client Area 登录账户
2. **进入 My Services**：顶部菜单 Services → My Services，查看所有 VPS 列表
3. **找到目标 VPS**：仔细核对，如果有多台 VPS，别搞错
4. **点击齿轮图标**：选择 Upgrade（升级）选项进入升级页面
5. **选择目标套餐**：页面会列出所有可升级的方案，包括套餐名称、价格和计费周期
6. **确认费用**：系统自动计算差价，核对金额是否合理
7. **完成支付**：支持支付宝、PayPal 等方式，Account Balance 账户余额也可以直接抵扣
8. **升级生效**：付款完成后立即生效，内存、硬盘、流量同步更新

升级全程不需要关机，IP 不变，数据不动。不过建议在操作前先在 KiwiVM 做一次快照备份，养成习惯，以防万一。

## 升级费用怎么算？不会浪费旧套餐剩余时间

很多人担心：我旧套餐还没到期就升级，之前付的钱不是亏了？不亏。

搬瓦工的差价计算方式：

> **升级费用 = (新套餐总价 - 旧套餐总价) / 付款周期天数 × 旧套餐剩余天数**

举个例子：年付 $49.99 的套餐，已经用了 60 天，想升级到年付 $99.99 的套餐。计算如下：

- 旧套餐剩余天数：365 - 60 = 305 天
- 每天差价：(99.99 - 49.99) / 365 = $0.137
- 实际需补交：305 × $0.137 ≈ **$41.8**

剩余时间全额折算成抵扣，不会白交。越早升级，需要补的差价越多；但如果快到期了再升级，差价少，不过也没多少时间享受新配置了。总体来说，按需升级就好，不必刻意选时机。

## FAQ：搬瓦工VPS升级常见疑问

**Q：升级之后 IP 地址会变吗？**
不会变。升级只是扩容配置，IP 保持不变，绑定的域名和服务不需要任何修改。

**Q：升级过程中 VPS 会停机吗？**
通常不会，升级速度很快，大多数情况下服务不会中断。建议如果网站对可用性要求高，选访问量低的时段操作。

**Q：升级之后能不能降回去？**
降级目前不支持自助操作，需要提交工单（Ticket）联系客服处理。客服是英文服务，工单里说清楚想降级到哪个套餐即可。

**Q：KVM 套餐能直接升到 CN2 GIA 吗？**
能。搬瓦工已经开放了从 KVM 跨系列升级到 CN2 GIA 的选项，在升级页面里能看到对应的选项，按正常流程选择就行。

**Q：升级时可以用优惠码吗？**
差价部分按比例适用优惠码折扣，如果当初购买时使用了折扣，升级时系统会自动按原折扣计算。

**Q：如果在升级页面看不到任何可选方案怎么办？**
第一步确认当前 VPS 状态是 Active（不是 Suspended）；第二步确认当前套餐不是该系列的最高配；如果排除以上两点还是没有选项，提交工单咨询客服，说明你想升级到哪个方案，让客服手动处理。
